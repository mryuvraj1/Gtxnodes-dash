## ✨ Features

### 🎮 **Game & Cloud Hosting**
- Minecraft Hosting (Budget & Premium Plans)
- Cloud VPS (Budget, Performance, Premium)
- Cloud VDS (Dedicated Resources)
- Domain Registration (100+ TLDs)

### 💰 **Billing & Payments**
- Razorpay Payment Integration
- Automatic Invoice Generation (PDF)
- Order Management System
- Live Currency Converter (INR, USD, PKR, EUR, JPY, BDT, AED, LKR, RUB)

### 👑 **Admin Panel**
- Dashboard Analytics (Users, Orders, Revenue)
- User Management (CRUD, Role Change)
- Plan Management (CRUD with Categories)
- Category & Subcategory Management
- Offer/Banner Management
- Dynamic Page Builder
- FAQ Management

### 👤 **User Dashboard**
- Profile Management
- Order History

- Service Status Tracking

### 🎨 **Design**

- Fully Responsive (Mobile/Desktop)

## 🚀 Quick Start

### Prerequisites

- Node.js 20+
- MongoDB 7+ (or MongoDB Atlas)
- Razorpay Account (for payments)

### Installation

```bash
# Clone repository
git clone https://github.com/yourusername/GTXNodes-Platform.git
cd GTXNodes-Platform

# Install backend dependencies
cd server
npm install

# Install frontend dependencies
cd ../client
npm install

# Create .env files (see below)
